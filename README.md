## 2 需求分析
> 这是软件开发流程的第二个阶段，也是为软件开发的正常进行确定具体思路的阶段。在确定软件开发可进行后，必须要对客户需要实现的软件功能需求进行具体详细的分析。同时应当考虑在开发过程中可能出现的变化情况，制定需求变更计划随时应对特殊情况的发生，保证软件开发流程的顺畅进行。
